Analisi dei Pattern

Fasi Evolutive

  • 2005-2006: Boom produttivo (962 file - periodo aureo)

Correlazioni Editoriali

Metodologia

I dati sono stati estratti direttamente dai metadati dei file dell'archivio digitale, analizzando le date di creazione e modifica per identificare pattern temporali e correlazioni editoriali.

Dataset Analizzato

  • 6,888 file con date di creazione verificate
  • 1,806 file con tracce di modifiche successive
  • Periodo: 1991-2022 (32 anni)
  • Formato: metadati temporali estratti automaticamente

Validazione

  • Cross-reference con cronologia editoriale
  • Verifica coerenza temporale
  • Controllo duplicati e anomalie
  • Correlazione con timeline biografica
